# Starter path

A step-by-step learning sequence for new NucleusIQ users.

## 1. Install and run

- [Install NucleusIQ](../install.md) — Core + your chosen provider (OpenAI or Gemini)

## 2. Add tools

- [Tools overview](../tools.md) — Three ways to create tools
- [`@tool` decorator](../tools/tool-decorator.md) — Create tools from plain functions (fastest path)

## 3. Choose an execution mode

- [Execution modes](../execution-modes.md) — Direct, Standard, Autonomous
- Start with **STANDARD** (the default) — add tools and let the agent iterate

## 4. Add memory

## 5. Track usage and costs

- [Usage tracking](../usage-tracking.md) — Token counts by purpose and origin

## 6. Try both providers

- [Gemini quickstart example](../examples/gemini-quickstart.md) — All three modes with Gemini
